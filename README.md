<h3 class="font-bold text-2xl">Description</h3>
<p>
  This project is a fully featured portfolio website template built in React.
  It is designed to help anyone quickly create and publish their own personal
  website with minimal setup. The template provides a clean structure, a modern
  layout, and easy customization so users can focus on presenting their work
  and identity.
</p>

<br/><br/>

<h3 class="font-bold text-2xl">Goals and Purpose</h3>
<p>
  - Provide an accessible starting point for people with little or no frontend experience.<br/>
  - Offer a modern, responsive design that works across all devices.<br/>
  - Allow users to customize their personal website by editing simple data files or React components.<br/>
  - Demonstrate how to structure a professional personal website using React.
</p>

<br/><br/>

<h3 class="font-bold text-2xl">Features</h3>
<p>
  <strong>Modern and Responsive UI</strong><br/>
  Clean and minimal design that works seamlessly on mobile, tablet, and desktop.
  <br/><br/>

  <strong>Built with React</strong><br/>
  Component-based architecture with reusable, modular code that is easy to extend.
  <br/><br/>

  <strong>Dynamic Project Viewer</strong><br/>
  Loads projects from JSON or local files, includes built-in filtering and search,
  and supports detailed project pages with images, videos, and descriptions.
  <br/><br/>

  <strong>Smooth Animations</strong><br/>
  Uses Framer Motion to create fluid section transitions and animated UI elements.
  <br/><br/>

  <strong>Easy Content Editing</strong><br/>
  Sections such as About Me, Projects, and Skills can be edited without any backend setup.
  <br/><br/>

  <strong>Hash-Based Routing</strong><br/>
  Fully compatible with static hosting platforms like GitHub Pages.
  <br/><br/>

  <strong>Lightweight and Fast</strong><br/>
  Optimized build, efficient asset loading, and minimal dependencies.
</p>

<br/><br/>

<h3 class="font-bold text-2xl">How To</h3>
<p>
<strong>Add Projects/Timeline:</strong><br/>
The website loads a JSON file from the web that stores all website data.  
This allows you to change the content of the website without touching its code.<br/>
The JSON file URLs can be found in: <code>src/constants/index.js</code> under the variables  
<code>PROJECTS_FILE_URL</code> and <code>TIMELINE_FILE_URL</code>.<br/>
Example files <code>Projects.json</code> and <code>Experience.json</code> are also included.
<br/><br/>

<strong>Change Colors</strong><br/>
To change the website's color theme, go to the root folder and open the <code>tailwind.config</code> file.  
This file controls all colors used in the website â€” from backgrounds to text.

</p>
